📜 История типографики

Типографика берёт начало с изобретения книгопечатания Иоганном Гутенбергом в XV веке. Первые шрифты имитировали рукописные тексты, но со временем развились в самостоятельную дисциплину.

В XIX веке появились гротески (шрифты без засечек), которые стали символом индустриальной эпохи. XX век принёс модульные шрифты и экспериментальные гарнитуры. Сегодня типографика — это не только красота, но и инструмент UX-дизайна.

Важно понимать, что каждый шрифт несёт эмоциональный заряд: серьёзные засечки — для классики, чистые гротески — для современности, рукописные — для тепла и уюта.

🏷️ Классификация шрифтов

С засечками (Serif)

Пример — Georgia
Алфавит: Аа Бб Вв Гг Дд
Используется в печатных изданиях, книгах, элегантных брендах.

Без засечек (Sans-serif)

Пример — Helvetica
Алфавит: Аа Бб Вв Гг Дд
Современный, нейтральный, для интерфейсов и плакатов.

Моноширинные (Monospace)

Пример — Courier New
Алфавит: Аа Бб Вв Гг Дд
Для кода, таблиц, технической документации.

Акцидентные (Display)

Пример — Impact
Алфавит: Аа Бб Вв Гг Дд
Для заголовков, акцентов, рекламы.

⚖️ Правила сочетания

Контраст

Сочетайте шрифты с разными начертаниями (например, засечный + гротеск) для создания визуального интереса.

Иерархия

Используйте разные размеры и веса для заголовков, подзаголовков и основного текста.

Ограничение

Не используйте более 2–3 шрифтов в одном макете — иначе будет хаос.

Настроение

Учитывайте характер шрифта: строгий или игривый, классический или футуристический.

Золотое правило: выберите один основной шрифт для текста и один акцентный для заголовков. Проверяйте читаемость на разных устройствах.
